## ğŸ§  How It Works

1. The user uploads a resume (PDF or text).
2. The agent analyzes:

   * Resume structure & formatting
   * Skill relevance and presentation
   * Clarity, grammar, and readability
3. The system assigns **scores** for each category.
4. Each score is **properly justified** using AI reasoning.
5. The final output includes:

   * Category-wise scores
   * Overall resume score
   * Clear explanations and improvement suggestions
